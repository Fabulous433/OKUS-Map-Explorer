# Attachments Smoke Checklist

## WP
- Upload PDF KTP/NIK pada dialog edit Wajib Pajak -> sukses
- Upload PDF NPWP pada dialog edit Wajib Pajak -> sukses
- Preview attachment gambar/PDF -> sukses
- Download attachment -> sukses
- Hapus attachment -> sukses

## OP
- Upload foto usaha pada dialog edit Objek Pajak -> sukses
- Upload foto lokasi pada dialog edit Objek Pajak -> sukses
- Upload PDF izin usaha pada dialog edit Objek Pajak -> sukses
- Preview attachment gambar/PDF -> sukses
- Download attachment -> sukses
- Hapus attachment -> sukses

## Validation
- Upload file `txt` ditolak dengan pesan `Format file tidak didukung`
- Upload file lebih besar dari `5 MB` ditolak dengan pesan `Ukuran file melebihi batas 5 MB`
