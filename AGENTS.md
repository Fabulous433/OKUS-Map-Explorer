# Global Agents for This Project

This project uses a shared global agent library from:

- `C:\Users\NB - MBA\.agents\agents`

## How to trigger

- If user mentions an agent by name (for example: `code-reviewer`, `backend-architect`, `frontend-developer`), load and follow that agent file from the global path above.
- If multiple agents are requested, use the minimal set and apply them in sequence.
- If an agent file is missing, state it briefly and continue with best-effort fallback.
